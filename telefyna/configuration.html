telefyna folder should be in USD/SDcard or if not existing Device Internal Storage where telefynaAudit folder containing audit messages are
Telefyna runs the scheduling at midnight daily;
This means if a program isn't in the folder before midnight it won't be scheduled or changes to config thereafter won't be picked up. To order Telefyna to re-run scheduling and use your changes afer the current program, add a file named init.txt in your telefynaAudit folder.
